网页中一些特殊的复杂效果,之前必须使用js,dom,jq 去编写
boot给我们封装了很多常用的组件, 可以快速的开发
问题:
1.由于boot封装的组件,很多自定义的效果,很难实现
2.关于事件,boot封装组件的事件,是使用自定义属性的方式实现的
使用boot封装的事件要注意:
1.激活事件的元素
2.事件影响的元素
1.按钮组
div.btn-group: 按钮组,横向
(1)弹性,x轴主轴,让内部btn挨在一起
(2)同时清空了,除了一个btn 以外的, 左上,左下圆角
(3)同时清空了,除了最后一个btn以外的, 右上,右下圆角
div.btn-group-vertical: 按钮组,垂直
(1)弹性,y轴主轴
(2)同时清空了,除了一个btn 以外的, 左上,右上圆角
(3)同时清空了,除了最后一个btn以外的, 左下,右下圆角
div.btn-group-sm/md/lg: 改变按钮组的大小, 修改内边距和字号

11.折叠导航栏
效果:
(1)最前面的菜单项,永远显示
(2)按钮: 大屏隐藏, 小屏显示
(3)导航: 大屏(显示,横向), 小屏(隐藏,纵向,需要点击按钮显示)
组件结构:
父亲div:
div.navbar
div.navbar-dark 让div内部所有文本变为浅色
div.bg-dark 配合navbar-dark 设置背景色
div.navbar-expand-md :
1.按钮在 md以上隐藏, sm显示
2.ul在 md以上横向, sm纵向
儿子:
第一部分:
div>a.navbar-brand 行内块,内边距
第二部分:
div>button.navbar-toggler 背景透明
div>button>span.navbar-toggler-icon 三条线
第三部分:
div>div.collapse 隐藏 源码--display: none
div>div.navbar-collapse 配合响应式,md以上显示,md一下隐藏
div>div>ul.navbar-nav 导航栏
div>div>ul>li.nav-item
div>div>ul>li>a.nav-link
事件:
1.激活事件的元素: button data-toggle="collapse"
2.事件影响的元素: data-target="#d1"
例:折叠导航栏

12.焦点轮播图
在一个父元素div中有三部分
(1)图片部分
(2)左右箭头
(3)轮播指示器
第一部分:图片
div#id = "demo"
div.carousel: 相对定位relative
div>div.carousel-inner: 相对定位,溢出隐藏 w-100
div>div>div.carousel-item: 相对定位,w-100,隐藏
事件:
data-ride="carousel"
第二部分:右箭头
a.carousel-control-next/prev
a>span.carousel-control-next-icon
事件:
data-slide="next"
data-slide="prev"
href = "#demo"; #demo最外层的id
注意: 需要重写样式
由于boot 封装好的样式不符合我们的需求,所以需要重写样式
第三部分:轮播指示器
ul.carousel-indicators 弹性, x轴主轴,居中,绝对定位
ul>li 父元素设置了li的样式
事件:
li: data-target="#demo", data-slide-to="图片编号" 编号由0开始
例:焦点轮播图
