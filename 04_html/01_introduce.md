## 2.web和internet
```
internet：全球性互联网，简称：互联网，交互网，交际网
www服务：world wide web 万维网服务
internet为web服务提供了网络环境
```
## 3.internet上所有程序的分类
```
C/S 结构  c:client  客户端   s:server 服务器   
B/S 结构  b：browser 浏览器   s:server 服务器   
c/s 结构，需要更新，更新的时候不能登录
b/s 结构，用户感受不到更新
```
## 4.web工作原理
```
web,运行在internet的一种服务应用程序，俗称网站
web的工作原理:
基于浏览器和服务器以及通信协议来实现数据的传输和展示

1.通信协议:
规范了数据是如何打包和传递的，好让前后台都认识这个数据,比如http协议
2.服务器:
  (1)功能
    存储数据
    接收用户的请求并给出响应
    提供后台程序的运行环境
    具备一定的安全性
  (2)产品
    TOMCAT(java), Apache（php）, IIS (.net)
  (3)技术
    PHP, JAVA, .net, python, nodejs
3.浏览器
  (1)功能
    发送请求
    接收响应
    把响应数据转换成可视化页面作为html,css,js的解析器
  2.产品
    chrome, safari, firefox, opera,IE
3.技术
    JavaScript,css,html
```
