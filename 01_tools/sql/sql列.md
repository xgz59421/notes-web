### 1. 列类型
```sql
创建数据表的时候，指定的列可以存储的数据类型
CREATE  TABLE  t1( nid 列类型 );
(1)数值型——引号可加可不加
  tinyint  微整型，占1个字节，范围-128~127
  smallint  小整型，占2个字节，范围-32768~32767
  int  整型，占4个字节，范围-2147483648~2147483647
  bigint   大整型，占8个字节，范围很大
  float  单精度浮点型，占4个字节，最大3.4E38，以牺牲小数点后n位为代价，精度有影响
  double  双精度浮点型，占8个字节
  decimal(M,D)   定点小数，小数点不会发生变化，几乎不会产生误差；M代表总的有效位数，D代表小数点后的有效位数
  boolean/bool   布尔型，只有两个值true(真)和false(假)；在真正使用过程中，自动转为tinyint，此时true转为1，false会转为0。 如果要使用true和false，因为是关键字不能加引号。往往存储一些只有两个值的数据，例如性别，是否在线，是否注册
(2)字符串型——必须加引号
  varchar(M)  变长字符串，几乎不会产生空间浪费，操作速度相对慢，M最大值是65535
  char(M)  定长字符串，可能产生空间浪费，操作速度相对快，M最大值是255，往往存储一些固定长度数据，例如手机号码，身份证号码等
  text(M)   大型变长字符串，M最大值是2G
(3)日期时间型——必须加引号
  date  日期型  2020-06-01
  time  时间型  15:25:30
  datetime  日期时间型    2020-06-01 15:25:30
(4)浮点型
  12345.6789
  1234.56789*10
  123.456789*10^2
  12.3456789*10^3
TB  GB  MB  KB  Byte  Bit（位）
1Byte=8Bit
位：二进制，只能存储0和1

CREATE  TABLE  t1(
  id  int,
  age  tinyint,
  phone  char(11),
  salary  decimal(7,2),    #99999.99
  sex  bool,
  content  varchar(3000),
  ctime  datetime
);
```
 
### 2. 列约束
```sql
mysql可以对要插入的数据进行特定的验证，只有符合条件才允许插入，否则被认为非法的插入，例如一个人的性别只能是男或者女，一个人工资只能是正数
CREATE  TABLE  t1(
  lid  列类型   列约束
);
#(1)主键约束——primary  key  
  声明了主键约束的列上禁止插入重复的值，一个表中只能有一个主键约束，通常加在编号列上，会加快数据的查找速度。
  NULL：表示空，在插入数据的时候，用来表示一个无法确定的值，例如无法确定商品的价格，员工的家庭住址...
  主键列是不允许插入NULL
#(2)非空约束——NOT  NULL
  声明了非空约束的列上禁止插入NULL
#(3)唯一约束 —— unique
  声明了唯一约束的列上不允许插入重复值，允许插入NULL，而且是多个NULL
  NULL是无法确定的值，未知的值；两个未知的值是不能划等号。
#(4)默认值约束 —— default
  可以使用default关键字来设置默认值，具体应用方式有两种
  INSERT  INTO  laptop  VALUES(5, '小米Pro', DEFAULT ...);
  INSERT  INTO  laptop(lid,title)  VALUES(6,'灵越');
#(5)检查约束 —— check
  检查约束也称为自定义约束
  CREATE  TABLE  t1(
    salary  DECIMAL(7,2)   CHECK(salary>=0)
  );
  mysql不支持检查约束，认为会对服务器造成比较大的压力，影响数据的插入速度。
#(4)外键约束——foreign key
  声明了外键约束的列，取值会到另一个表的主键列中找；允许插入NULL，一个表中可以有多个外键约束；外键和对应的主键列类型要保持一致。
  foreign key(外键列)  references  另一个表(主键列)
#(5)自增列
  auto_increment: 自动增长，在插入数据的时候，只需要设置值为NULL，就会获取当前的最大值然后加1插入。
  注意：
  自增列必须添加在主键列上
  自增列允许手动赋值
  主键是不允许设置为NULL，但是添加了自增列，要想实现自增又允许使用NULL
  ```
