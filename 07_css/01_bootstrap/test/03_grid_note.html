1.web 页面常用的布局方式一般3种

语义错误,渲染效率低下

(1)row自带的左右-15的外边距,会让row 变宽
(2)col自带的左右15px内边距,会影响col的布局
(3)向上兼容
(4)通常清除掉: no-gutters
