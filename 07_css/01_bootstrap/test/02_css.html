1.全局css样式

(1)容器 container:
1.boot不支持超小屏, (支持 xl lg md sm)
2.所有的boot都要写在container中
3.定宽容器,不同屏幕下,有写死的最大宽度
4.boot中 1rem = 16px
container-fluid 变宽容器

(2)按钮相关样式
btn: 基本类 (设置了 行内块, 内边距, 文本,过渡 等等)
按钮的颜色: btn-success/danger/warning/primary/secondary/info/light/dark....
带边框的按钮: btn-outline-danger/warning......
btn-lg: 大按钮
btn-sm: 小按钮, 通过修改内边距,字号,行高,来实现不同大小
btn-block: 块级按钮 width:100%

(3)图片相关
1.rounded: 圆角 0.25rem
2.rounded-circle: 圆
3.img-thumbnail: 缩略图, 有一圈白色内边, 有边框
4.img-fluid: 响应式图片, 图片可以缩放,但是不会超过原始尺寸

(4) 文本相关
1.文本颜色: text-danger/warning/info/ muted (灰色) ....
2.文本大小写:
text-uppercase: 大写
text-lowercase: 小写
text-capitalize: 首字母大写
3.字体粗细:
font-weight-bold: 粗
font-weight-normal: 正常
font-weight-light: 细
4.字号大小: h1~h6
5.文本水平对齐方式:
text-left/center/right:
text-justify: 两端
6.文本水平对齐方式:(响应式)
1.text-xl-left/center/right
2.text-lg-left/center/right
3.text-md-left/center/right
4.text-sm-left/center/right
注意:
1.响应式的兼容, 屏幕向上兼容
2.响应式的文本对齐, 没有justify,手写justify媒体查询
7.文本溢出变为小点点: text-truncate
